# My Blog

A simple, clean blog built with HTML, CSS, and vanilla JavaScript.

## Features

- Clean, responsive design
- Easy to customize
- No build tools required
- Lightweight and fast

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/my-blog.git
   cd my-blog
   ```

2. Open `index.html` in your browser or deploy to Cloudflare Pages

## Deployment to Cloudflare Pages

1. Push this repo to GitHub
2. Go to [Cloudflare Pages](https://pages.cloudflare.com/)
3. Connect your GitHub account and select this repository
4. Click "Save and Deploy"

Your blog will be live in minutes!

## Adding New Posts

Create new HTML files in the `posts/` directory following the existing post structure.

## Customization

- Edit `style.css` to change colors and layout
- Update the header and footer in `index.html`
- Modify the color variables in `:root` in `style.css`
